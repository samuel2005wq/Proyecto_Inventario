\doxysubsection{Detailed Description}
Representa la pestaña de interfaz gráfica para la gestión de inventario. 

Permite al usuario visualizar, agregar, modificar y eliminar componentes del inventario mediante una tabla y un formulario. Se conecta a la lógica de inventario a través de \doxylink{classInventoryManager}{Inventory\+Manager}.
