Detailed Description

Clase que maneja la lógica de inventario utilizando una base de datos.

Clase que gestiona la lógica del inventario, interfaciando con DatabaseManager.

Esta clase proporciona funciones para agregar, consultar, buscar, actualizar y eliminar componentes en un inventario. Internamente utiliza una instancia de DatabaseManager para manejar operaciones relacionadas con la base de datos.
